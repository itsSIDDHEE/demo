Deploy Web App (nginx) via Docker CLI

Step 1: Pull nginx Image
Bash
docker pull nginx:latest

Step 2: Run Container Interactively
This gives you a shell inside the container and maps port 5555.

Bash
docker run -it -p 5555:80 --name server_ismaeel nginx:latest bash

Step 3: Inside Container - Modify Webpage
You are now root inside the container.

Bash
# Go to web directory
cd /usr/share/nginx/html/

# Backup default page
mv index.html index.html_backup

# Create new panano index.htmlge
nano index.html

code:

<!DOCTYPE html>
<html>
<head>
    <title>Welcome to ismaeel's Server!</title>
    <style>
        body { 
            font-family: Arial, sans-serif; 
            display: grid; 
            place-items: center; 
            min-height: 90vh; 
            background-color: #f4f4f4; 
        }
        h1 { 
            color: #333; 
            border-bottom: 2px solid #0096d6; 
            padding-bottom: 10px;
        }
    </style>
</head>
<body>
    <h1>Hello from my custom Nginx container!</h1>
</body>
</html>

Save & Exit: Ctrl+O, Enter, Ctrl+X.

Step 4: Inside Container - Start nginx Service
Bash

# Check status (will show not running)
service nginx status

# Start nginx
service nginx start
Step 5: Verify
Open your host machine's browser.

Go to http://localhost:5555.
